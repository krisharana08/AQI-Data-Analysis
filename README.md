# 📊 AQI Data Analysis Project

## 🔹 Objective
To analyze Air Quality Index (AQI) data and identify pollution trends, seasonal patterns, and key environmental insights.

## 🔹 Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  

## 🔹 Key Steps
- Data Cleaning  
- Exploratory Data Analysis (EDA)  
- Data Visualization  

## 🔹 Key Insights
- Identified cities with highest pollution levels  
- Observed seasonal trends in AQI  
- PM2.5 found as a major pollutant  

## 🔹 Conclusion
This project highlights how data analysis can help understand environmental conditions and support decision-making for pollution control.
